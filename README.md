## インストール

[Releases](https://github.com/sevenc-nanashi/bpm_object.aux2/releases/latest)から`sevenc-nanashi.bpm_object-aux2-v<バージョン>.au2pkg.zip`をダウンロードし、AviUtl2のプレビューにドラッグ&ドロップしてください。
